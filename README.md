All just a after-school research for Arduino and XBee S2C.

Use Components:
LCD screen : "DFROBOT LCD KEYPAD SHIELD V1.1"
XBee : "XBee S2C"

X-CTU setting :
<coordinator>
  ID PANID : 1001
  SC Scan Channels: 7FFF
  AP API Enable : 1(Enable)
  
<router>
  ID PANID : 1001
  SC Scan Channels: 7FFF
  AP API Enable : 0(Transparent Mode)
